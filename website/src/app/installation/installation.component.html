Installation

  • Install this extension from the Edge web store or from the source
  • Enter DevTools (using F12 or Ctrl + shift + i) and go to settings by pressing "?" (Shift + /).
  • Under "Appearance", change the theme to "Dark."
  • Go to experiments in the left panel, and enable "Allow custom UI themes"
  • Restart DevTools.

How to install from source

  • Go to the projects GitHub and download the zip file (direct link)
  • Extract the folder
  • Head to edge://extensions or edge://extensions.
  • Go to the bottom left corner and enable 'Developer Mode'
  • Drag and drop the 'extension' folder to any part of that tab.
